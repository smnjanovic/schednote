setNote

@StringRes fun setNote(context: Context, note: Note, onSuccess: (Note) -> Unit): Int?

Pokúsi sa vytvoriť úlohu, uložiť ju do databázy a nastaviť k nej pripomienku, ak sú upozornenia na pripomienky zapnuté.

Parameters

context - Ľubovoľný kontext

note - Úloha, ktorú sa snažím nastaviť

onSuccess - Funkcia, ktorá sa má vykonať po úspešnom nastavení úlohy

Return
Po úspešnom nastavení úlohy vráti null, inak vráti odkaz na text popisujúci dôvod neúspechu vytvorenia úlohy.

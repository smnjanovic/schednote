restore

protected fun <T> restore(key: String, default: T? = null): T?

Zjednodušené ukladanie hodnôt typu T. Typ T nesmie byť nič iné ako primitívny typ Reťazec alebo rad reťazcov (Set).

Parameters

key - značka pod ktorou sa hodnota nájde

default - predvolená hodnota, ak pod značkou key žiadna hodnota neexistuje

Exceptions

ClassCastException - Príčiny: Keď nenulová hodnota default je iných typov, ako: Boolean, Float, Int, Long, String, Set Keď je pod kľúčom key vložená hodnota iného typu ako T

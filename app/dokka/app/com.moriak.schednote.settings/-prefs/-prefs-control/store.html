store

protected fun <T> store(key: String, value: T?): Unit

Zjednodušené ukladanie hodnôt typu T. Typ T nesmie byť nič iné ako primitívny typ Reťazec alebo rad reťazcov (Set).

Parameters

key - značka pod ktorou sa hodnota uloží

value - Vkladaná hodnota typu T. Keď je null, prvok s označením key bude odstránený

Exceptions

ClassCastException - Keď nenulová hodnota value je iných typov, ako: Boolean, Float, Int, Long, String, Set.

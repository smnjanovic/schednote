unsafeRestore

protected fun <reified T> unsafeRestore(key: String): T

Získava údaj z trvalého úložiska aplikácie typu T pod kľúčom key. Chyba, ktorá nastane, keď pod kľúčom T je uložený údaj iného typu ako T, tu nie je ošetrená!

Parameters

key - Kľúč, pod ktorým je údaj dostupný

Exceptions

ClassCastException - pod kľúčom T je uložený údaj typu iného ako T.

InvalidTypeException -

zo získaných dát nie je možné vytvoriť údaj typu T. typ T môže zastupovať typy Boolean, Int, Float, Long, String, Uri, Set a Enum.

Pre úspešnosť zísania hotnoty typu enum je nutné do atribútu enumMap uložiť pole inštancií triedy typu T zastupujúcej Enum pod kľúčom v tvare jej kanonického názvu Class.getCanonicalName.

Return
vráti uložený údaj alebo predvolenú hodnotu typu T
